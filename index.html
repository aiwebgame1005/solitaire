How to Play Solitaire

Game Objective

Move all cards to the four foundation piles, building up from Ace to King in each suit.

Card Arrangement

Build descending sequences of alternating colors in the tableau columns.

Empty Columns

Only Kings can be placed on empty tableau columns to start new sequences.

Solitaire Variations

Klondike

The classic version most people recognize as traditional Solitaire.

Spider Solitaire

A more complex variation using two decks of cards.

FreeCell

A strategic version where almost every game is solvable.

Pyramid

Cards are arranged in a pyramid shape, pairs adding up to 13 are removed.

Golf

A fast-paced variation where cards can be removed in sequence.

Yukon

Similar to Klondike but with different tableau building rules.
